## General info
This is Turkish Checkers Game. You can play this game with your friend.
	
## Technologies
Project is created with:
* JavaScript
* SCSS
* react: 18.2.0
* redux: 4.2.0
* react-redux: 8.0.2
* reduxjs/toolkit: 1.8.5

	
## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
